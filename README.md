Create `.env` file and fill with below data:
```bash
DB_USER='DATABASE_USERNAME'
DB_HOST='DATABASE_HOSTNAME'
DB_NAME='DATABASE_NAME'
DB_PASS='DATABASE_PASSWORD'
DB_PORT=DATABASE_PORT

PORT=RUNNING_PORT
```
